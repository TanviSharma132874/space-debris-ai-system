## Getting Started

### Prerequisites

- Docker
- Docker Compose
- Node.js
- npm

### Installation

1. Clone the repository.
2. Create a `.env` file in the `server` directory with your MongoDB connection string and a JWT secret.
3. Run `docker-compose up --build` to start all services.

- **Frontend:** `http://localhost:5173`
- **Backend:** `http://localhost:5000`
- **MongoDB:** `mongodb://localhost:27017`
